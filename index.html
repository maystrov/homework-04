git --version
Проверка установлен ли Git и его версию
git init
Инициализация нового репозитория. Один раз для проекта.
git add index.html
Добавляем к коммиту файл index.html
git commit -m "Создал файл "index.html"
Создание нового коммита (новой записи в истории ре)
git status
Показывает состояние текущего репозитория
git remote add origin https://github.com/maystrov/Lesson-4.git
Добавляем связь между удаленным и локальным репозиторием
git push origin master
Отправляем наши изменения в удаленный репозиторий
